REST API

검색서버와의 통신을 HTTP REST 방식을 사용하므로, 클라이언트의 제약이 없습니다. 또한 결과포맷으로 JSON과 XML을 지원합니다.

  • 검색어 입력 REST API
  • 인기검색어 REST API
  • 연관검색어 REST API
http://localhost:8050/service/keyword/popular/rt.xml?siteId=total&categoryId=_root
<?xml version="1.0" encoding="UTF-8"?>
<response>
  <siteId>total</siteId>
  <categoryId/>
  <list>
    <item>
      <rank>1</rank>
      <word>해리포터</word>
      <diffType>EQ</diffType>
      <diff>0</diff>
      <count>89</count>
      <countDiff>0</countDiff>
    </item>
    ...
  </list>
</response>

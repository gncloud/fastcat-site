fastcatsearch | 실시간 검색엔진

실시간 검색

변경되는 문서에 대해 즉시 색인되어 검색결과로 나타납니다.

  • 데이터베이스 내부문서검색
  • 워드, 한글, PDF 첨부파일 문서검색
  • 다중정렬, 다중필터링
  • 패싯(그룹핑)검색

분산시스템

데이터를 여러 분산노드에 색인함으로써 대용량처리와 빠른 병렬검색이 가능합니다.

  • 분산노드구성
  • 색인데이터 파티셔닝
  • 병렬 검색 및 검새결과 머징
  • 분산노드관리

다양한 검색연산지원

불리언검색을 기반으로 완전매칭뿐 아니라, 여러가지 검색연산을 지원합니다.

  • 불리언검색
  • 키워드 완전매칭검색
  • 전방일치검색
  • 인접검색
  • 부스팅검색

정렬, 필터링, 패싯(그룹핑)

정렬, 필터링, 패싯은 검색품질을 높이는데 필수적입니다.

  • 정렬 - 다중정렬, 정수형/문자형 필드정렬
  • 필터링 - 다중필터링, 정수형/문자형 필터링, 부스팅필터링
  • 패싯 - 필드그룹핑, 여러필드 그룹핑

문서파일검색

데이터베이스이외에도 MS오피스, 한글(HWP), PDF등의 문서파일내용도 검색이 가능합니다.

  • MS오피스문서 - DOCX, PPTX, XLSX
  • 한컴오피스 - HWP
  • PDF
  • AUTOCAD문서 - DWG, DXF
  • 그외의 다양한 문서포맷 지원

사전관리

검색품질을 높이기 위해 기본적으로 사용자, 유사어, 금지어사전을 지원하며, 운영자만의 커스텀사전을 만들어서 관리할수도 있습니다.

  • 사용자사전 - 신조어에 대응
  • 유사어사전 - 유사단어검색
  • 금지어사전 - 금지단어제외
  • 커스텀사전 - 운영자만의 사전관리

REST API

검색서버와의 통신을 HTTP REST 방식을 사용하므로, 클라이언트의 제약이 없습니다. 또한 결과포맷으로 JSON과 XML을 지원합니다.

  • 검색 REST API
  • 색인 REST API
  • 관리 REST API

운영자 계정/그룹관리

운영자 그룹별로 관리권한을 제공하므로써 불필요한 권한남용을 막을 수 있습니다.

  • 그룹별 권한관리
  • 그룹별 계정관리
  • 그룹정보관리
  • 계정정보관리

완벽한 모바일 지원

모바일에서도 검색기능을 제공할 수 있습니다. 또한 HTML5기반의 관리도구를 제공함으로써 모바일환경에서도 동일하게 검색운영을 할 수 있습니다.

  • 모바일 검색기능
  • HTML5기반 웹관리도구

  • 색인주기설정

    반복적인 증분색인 주기를 운영자가 원하는대로 설정함으로써 운영환경에 적합한 스케쥴링이 가능합니다.

  • 데이터 플러그인

    문서를 색인시 데이터소스 확장기능을 사용하면, 자체적으로 플러그인을 작성하여, 어떤한 포맷의 문서라도 검색이 가능합니다.

  • 분석기 플러그인

    기본적으로 제공되는 언어분석기 이외에도, 외부 언어분석기를 플러그인으로 래핑하여 사용할 수 있습니다.
